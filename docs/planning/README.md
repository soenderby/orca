# Orca Planning Documents

These files describe proposed redesign and expansion work.
They are intentionally separated from runtime docs so implementation plans do not get mixed with current behavior.

## Files

- `orca-redesign.md` — redesign proposal and design decisions
- `orca-roadmap.md` — milestone-based implementation roadmap
- `orca-goals-and-considerations.md` — broader goals, principles, and layer model

## Status Guidance

- Planning docs may be internally inconsistent while exploration is ongoing.
- Runtime behavior is defined by the root operational docs and shell scripts.
